<!-- GETTING STARTED -->
## Getting Started

### Prerequisites

  ```sh
  python3.12 -m venv venv
  source venv/bin/activate
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/ClemouDouard/skaizen-webscraping.git
   ```
2. Install packages
   ```sh
   pip install -r requirements.txt
   ```
3. Get your api keys
   1. Go to those websites
     - https://serper.dev/
   2. `touch .env`
   3. Put your keys in there
